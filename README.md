## 프로젝트 소개

- 서비스명 : 혼자 뭐하니?
- 소재 : 팀 매칭과 SNS
- 원하는 그룹을 찾고 서로의 운동욕구를 자극하는 SNS 프로그램
  - 원하는 그룹을 찾아 가입할 수 있다.
  - 그룹을 통해 팀원들과 같이 운동할 수 있는 기회를 제공한다.
  - SNS를 통해 서로 소통하며 첼린지를 통해 서로의 운동욕구를 자극한다.
  - 결재시스템과 추천시스템을 활용해 차별화된 서비스를 제공한다.
